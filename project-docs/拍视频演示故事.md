# 拍视频演示故事 - 《迷路的小猫》

**创建时间**: 2026-04-29 19:29  
**情绪类型**: 6 种（无开心）  
**情绪起伏**: 大

## 📖 完整故事（可直接输入）

```
傍晚时分，阳光透过树叶洒在地上，我像往常一样在小区里散步，一切都那么宁静祥和。

突然，草丛里传来微弱的叫声，我拨开草丛，发现一只瑟瑟发抖的小猫，它的腿上缠着 broken 的绳子。

小猫的眼睛里满是恐惧和无助，身上脏兮兮的，显然已经流浪很久了。看着它瘦弱的样子，我心里一阵酸楚。

是谁这么狠心抛弃了它？绳子勒得那么紧，一定很疼。想到有些人对生命的冷漠，我感到非常愤怒。

我小心翼翼地抱起小猫，带它回家清理伤口、喂食。它慢慢放松下来，用头蹭我的手，那一刻我感到被信任的温暖。

晚上小猫突然呼吸困难，我慌了神，担心它撑不过去，手忙脚乱地联系兽医，心跳得厉害。

兽医说只是惊吓过度，休息几天就好。看着小猫安稳地睡着，我终于放下心来，窗外月色正好。
```
