# 动作变体映射表

**用途**：根据情绪 + 风格生成不同的动作变体，避免重复

## 风格选择逻辑

```python
def select_action_style(emotion, style, intensity=None):
    """根据情绪和风格选择动作序列"""
    
    ACTION_MAP = {
        "happy": {
            "default": [1, 6, 0],
            "formal": [1, 0],
            "warm": [1, 6, 0],
            "climax": [1, 6, 7, 0],      # 加入自由飞翔
            "gentle": [1, 0],
            "transition": [1, 9, 0],
        },
        "grateful": {
            "default": [3, 4, 0],
            "formal": [3, 4, 0],         # 完整致谢
            "warm": [3, 1, 0],           # 致谢 + 挥手
            "climax": [3, 4, 1, 6, 0],
            "gentle": [3, 0],
            "transition": [4, 1, 0],
        },
        "scared": {
            "default": [8, 10, 0],
            "formal": [8, 10, 0],
            "warm": [8, 0],
            "climax": [10, 8, 10, 0],
            "gentle": [10, 0],
            "transition": [8, 1, 0],
        },
        # ... 其他情绪
    }
    
    # 获取基础序列
    style_map = ACTION_MAP.get(emotion, {})
    sequence = style_map.get(style, style_map.get("default", [0]))
    
    # 根据强度微调
    if intensity:
        if intensity < 0.4 and style != "climax":
            # 低强度：简化动作
            sequence = sequence[:2] if len(sequence) > 2 else sequence
        elif intensity > 0.8 and style == "climax":
            # 高强度 + 高潮：确保使用最丰富序列
            pass  # 保持原序列
    
    return sequence
```

## 使用示例

### 示例 1：重复情绪避免重复动作

```python
# 小明的故事
segments = [
    {"emotion": "happy", "style": "default"},     # 段落 1
    {"emotion": "grateful", "style": "formal"},   # 段落 2：首次感谢
    {"emotion": "scared", "style": "climax"},     # 段落 3：高潮
    {"emotion": "grateful", "style": "warm"},     # 段落 4：再次感谢（变体！）
    {"emotion": "happy", "style": "climax"},      # 段落 5：结尾高潮
]

# 生成动作序列
actions = []
for seg in segments:
    action = select_action_style(seg["emotion"], seg["style"])
    actions.append(action)

# 结果
[
    [1, 6, 0],      # happy-default: 挥手→舞蹈
    [3, 4, 0],      # grateful-formal: 致谢→致谢
    [10, 8, 10, 0], # scared-climax: 后退→下蹲→后退
    [3, 1, 0],      # grateful-warm: 致谢→挥手 ✅ 不同！
    [1, 6, 7, 0],   # happy-climax: 挥手→舞蹈→自由飞翔 ✅ 高潮！
]
```

### 示例 2：情绪曲线应用

```
情绪曲线：rising → peak → falling

段落 1 (rising):   happy + formal    → [1, 0]       克制开场
段落 2 (rising):   grateful + warm   → [3, 1, 0]    温暖上升
段落 3 (peak):     scared + climax   → [10,8,10,0]  恐惧高峰
段落 4 (falling):  grateful + gentle → [3, 0]       轻柔回落
段落 5 (end):      happy + climax    → [1,6,7,0]    兴奋结尾
```
